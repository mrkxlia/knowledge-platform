# AI Knowledge Platform

A personal AI Knowledge Platform for learning AWS, AI application development, and software architecture.

## 🎯 Goals

This project is built to:

- Learn AWS through hands-on development
- Learn AI application development
- Build a long-term personal knowledge platform
- Practice maintainable software architecture

## 🚀 Features (Planned)

- Markdown Notes
- Full-text Search
- AI Search
- AI Chat
- Prompt Management
- Project Management
- RAG
- MCP Server
- AI Agent Integration

## 🏗️ Tech Stack

### Backend

- Python
- FastAPI

### Frontend

- React
- TypeScript

### Infrastructure

- AWS
- Docker
